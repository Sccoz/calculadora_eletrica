# Conversor de Unidades Elétricas

Este é um script simples em JavaScript que permite ao usuário converter diferentes unidades elétricas, como corrente elétrica, tensão elétrica, resistência elétrica e potência elétrica, entre diferentes prefixos.

## Funcionalidades

- O usuário pode escolher a unidade elétrica que deseja converter (Ampère, Volt, Ohm ou Watt).
- O usuário pode escolher o prefixo do valor de entrada (Giga, Mega, Quilo, base, Mili, Micro ou Nano).
- O usuário pode escolher o prefixo para o qual deseja converter o valor (Giga, Mega, Quilo, base, Mili, Micro ou Nano).
- O usuário pode inserir a quantidade que deseja converter.
- O script calcula o valor na base e exibe o resultado final com o prefixo escolhido.

## Como usar

1. Copie o código fornecido.
2. Crie um novo arquivo HTML e cole o código JavaScript dentro da tag `<script>`.
3. Abra o arquivo HTML em um navegador.
4. Siga as instruções exibidas na página para realizar as conversões.
